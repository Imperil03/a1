# a1

Стартовый репозиторий для сайта digital-agency.

Направление: чистый premium-tech стиль для агентства, которое делает контекстную рекламу, SEO, сайты, аналитику и мобильные приложения.

## Референс

- `references/reference-brief.md` - что берем из присланного скрина и как адаптируем под агентство.
- `references/agency-hero-reference.html` - standalone HTML-референс первого экрана.

## Важно

Исходный файл картинки из чата недоступен как файл на диске, поэтому в репозиторий добавлен текстовый и HTML-референс по присланному изображению. Когда появится PNG/JPG, его можно положить в `assets/reference.png`.
